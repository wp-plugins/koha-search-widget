== Description ==

A tiny widget that allows you to easily add a search box for your Koha catalog, as long as you know the URL of your Koha OPAC.
== Installation ==


1. Use the built in plugin uploader to upload the plugin to your installation
1. Activate the plugin through the 'Plugins' menu in WordPress (or click the link after install)
1. Go to Appearance -> Widgets to add the widget to your sidebar.

== Frequently Asked Questions ==

= What's the deal with the Title text? =

The title text is arbitrary, and will be the header of the widget.

= Are there any restrictions on the URL to my Koha Catalog? =

Only that there are no trailing slashes, and you must have http:// at the beginning.
